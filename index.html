Doomenstein - DX11 3D - School Composition Award

2024

A first-person shooter inspired by Wolfenstein, built entirely on DirectX 11 and WinAPI. The 3D environment is structured using a custom grid-based level definition, with geometry dynamically generated from the map data.

Key implemented systems include:

  • AI & Navigation: Implements the A* pathfinding algorithm over the level grid, augmented with path smoothing and dynamic path recalculation. Features two distinct raycasting modes for line-of-sight checks and player interaction detection. Enemy behavior is managed by a State Machine controlling Idle, Move, and Attack sequences.
  • Collision & Physics: Collision logic uses AABB vs. Sphere checks for player/enemy movement, including custom logic for smooth corner traversal. Enemy-to-enemy overlap is resolved by a Safe Separation routine utilizing AABB vs. AABB checks.
  • Rendering & Animation: Features GPU-side Lambertian lighting and incorporates billboarding techniques for all object and enemy sprites. A custom sprite system manages enemy and player weapon animations across multiple defined states.
  • Tooling & Audio: Developed a custom asset packer to manage and load texture assets. Audio is integrated via XAudio2.
